Desenvolvi um Sistema de Gerenciamento de Eventos para Igreja com o objetivo de organizar e administrar os eventos realizados pela instituição de forma prática e eficiente.

O sistema permite realizar operações completas de CRUD:
  - ✅ Cadastrar eventos
  - 📋 Listar eventos cadastrados
  - ✏️ Editar informações dos eventos
  - ❌ Excluir eventos


🛠️ Tecnologias Utilizadas
  - Java – Linguagem principal do projeto
  - MySQL – Banco de dados relacional
  - JDBC – Conexão com o banco de dados
  - Programação Orientada a Objetos (POO)
  - Tratamento de Exceções para maior segurança e estabilidade da aplicação


Durante o desenvolvimento, foram aplicados conceitos importantes como:
  - Encapsulamento
  - Separação de responsabilidades
  - Manipulação de banco de dados com SQL
  - Tratamento de erros com try-catch
  - Validação de dados


O sistema foi criado para facilitar o controle de eventos da igreja, evitando organização manual e reduzindo erros no registro de informações.

Além disso, o projeto teve como foco o aprimoramento das habilidades em:
  - Integração Java + MySQL
  - Estruturação de projetos reais
  - Aplicação prática de POO
  - Desenvolvimento de sistemas com persistência de dados
